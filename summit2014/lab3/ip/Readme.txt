Home Loan - Information Processing Definitions
=====================================================

This quickstart provides an example configuration for information processors related to the messages sent and received by the Order Management application. These information processors are responsible for extracting information that may be required in subsequent event processing.

To deploy the information processor, after the server has been started, run:

	mvn jboss-as:deploy

To undeploy the quickstart, run:

	mvn jboss-as:undeploy
